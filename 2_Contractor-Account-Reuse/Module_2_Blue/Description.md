# Challenge: Contractor Account Reuse in the Control Room (Blue Team)

**Background:**
You are an OT Security Analyst operating the SOC for a regional electrical substation. It's Monday morning, and an automated rule has flagged anomalous off-hours activity against your Rapid SCADA Webstation.

**Your Objective:**
You have been provided with the raw SCADA application logs and the system maintenance records. You must conduct a forensic review to identify the scope of the breach. You need to prove *who* logged in, *when*, from *where*, *what* they accessed, and *why* they still had access in the first place.

* **Framework Mapping:** MITRE ATT&CK for ICS
* **Tactics:** TA0110 (Persistence), TA0109 (Lateral Movement)
* **Technique:** T0859 (Valid Accounts)
